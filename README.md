# CPU Scheduling Simulator

A web-based CPU Scheduling Simulator developed using HTML, CSS, JavaScript, and Tailwind CSS. The project demonstrates the working of different CPU scheduling algorithms with Gantt chart visualization and process scheduling calculations.

---

## Features

- FCFS (First Come First Serve)
- SJF (Shortest Job First)
- Priority Scheduling
- Round Robin Scheduling
- Gantt Chart Visualization
- Waiting Time Calculation
- Turnaround Time Calculation
- Responsive User Interface

---

## Technologies Used

- HTML
- CSS
- Tailwind CSS
- JavaScript

---

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Run the `index.html` file in a browser
